This project requires Intellij IDEA and JDK 17
GitHub repository: https://github.com/npkasenides/double-project.git

Email dataset adapted from: https://www.kaggle.com/datasets/balaka18/email-spam-classification-dataset-csv
Filler words adapted from: https://www.geeksforgeeks.org/removing-stop-words-nltk-python/

This project can be opened in Intellij using the "double-project.iml" file
It can then be run using the Menu Bar (Run > Run 'MainForm')
